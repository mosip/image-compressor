# image-compressor
Java based simple JPEG2000 image compressor. Supports Both android and linux.

## Modules

The repository encompasses JPEG2000 image compressor. Below is an overview of each module, along with links to their respective internal README files for detailed information:

### 1. MockMDS
This module simulates the JPEG2000 image compressor. [Read more](https://github.com/mosip/image-compressor/blob/master/image-compressor/README.md)
